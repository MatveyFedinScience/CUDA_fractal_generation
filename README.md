## Что такое фрактал?


**Фрактал** — это геометрическая фигура, обладающая свойством самоподобия. Это означает, что она составлена из нескольких частей, каждая из которых подобна всей фигуре целиком. Фракталы часто имеют дробную метрическую размерность и описываются рекурсивными алгоритмами.


### Множество Мандельброта


Множество Мандельброта — это множество точек $c$ на комплексной плоскости, для которых итеративная последовательность не уходит в бесконечность.

Основная формула итерации:
$$z_{n+1} = f(z_n) + c$$
где $z_0 = 0$. Если модуль $|z_n|$ остается ограниченным при $n \to \infty$, то точка $c$ принадлежит множеству.


### Множество Жюлиа


Множество Жюлиа тесно связано с множеством Мандельброта. Оно строится по той же формуле:
$$z_{n+1} = f(z_n) + c$$
Однако, здесь $c$ — это **фиксированный** комплексный параметр, а итерации начинаются для различных начальных точек $z_0$ на плоскости. Форма множества Жюлиа зависит от того, какую константу $c$ вы выберете.


## Цель проекта


Этот проект разработан для выскэффективной генерации фракталов с использованьем C/C++ и CUDA. Особое внимание уделено пакетной генерации и в проекте сделан акцент на генерацию фракталов из определённого класса. За функцию $f$ взят отрезок ряда Лорана, максимальная и минимальная степень которого указывается в `config.h`.


## Конфигурация


Файл `config.h` содержит все параметры, которые управляют генерацией фрактала. Указанная там в данный момент конфигурация протестирована в тестовом окружении (see System Specifications).

## Using


Перед использованием проект необходимо скомпилировать командой:

```bash
./compile.sh
```

После этого вы можете запустить генерацию фракталов. Поддерживаются 3 варианта использования:


```bash
  ./mandelbrot                       # Random fractals  
  ./mandelbrot data.csv              # Generate from CSV
  ./mandelbrot a.csv -c b.csv        # Compare two CSVs
```

Где режим `-c` выводит в консоль отношение симметрической разности к объединению в среднем по фракталам с параметрами из двух файлов. В данный момент это поддерживает не более 128 фракталов в каждом из сравнивемых множеств, в будующем это ограничение планируется снять.

В стандартном режиме в файле output.csv вы найдете параметры $f(z)$ в разложении ряда Лорана, а в папке output/ вы найдете сгенерированные фракталы.

В режиме генерации из файла сохраняются только картинки.
